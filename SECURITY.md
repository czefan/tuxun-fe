# 安全策略

## 支持的版本

我们积极为以下版本的 **tuxun-fe** 提供安全更新与缺陷修复支持：

| 版本     | 是否支持安全更新        |
| -------- | ----------------------- |
| >= 1.0.0 | :white_check_mark: 支持 |
| < 1.0.0  | :x: 不支持              |

## 报告安全漏洞

如果您在本项目中发现了潜在的安全漏洞，请**不要**直接创建公开的 Issue。直接公开漏洞可能会导致漏洞在修复被发布前被恶意利用。

相反，请通过 GitHub 仓库原生的 **[私密漏洞报告功能 (Private Vulnerability Reporting)](https://docs.github.com/cn/code-security/security-advisories/guidance-on-reporting-and-writing-vulnerabilities/about-private-vulnerability-reporting)** 提交您的安全缺陷报告。

或者，您可以通过查阅项目的 Git 提交历史或维护者资料中的公开邮箱与我们取得联系。

非常感谢您协助我们一起维护该项目的安全！
